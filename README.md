# Ninos Kebab Demo

Detta är en körbar demo av Ninos Kebab beställningsapp.

## Kör lokalt

```bash
npm install
npm run dev
```

Öppna sedan `http://localhost:3000`.

## Publicera på Vercel

1. Skapa konto på vercel.com
2. Skapa nytt projekt
3. Ladda upp denna mapp eller koppla GitHub
4. Klicka Deploy

## Ingår

- Kundvy
- Varukorg
- Leverans/hämtning
- Simulerad betalning
- Köksskärm
